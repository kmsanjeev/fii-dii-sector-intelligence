# ADR-011

## Title

Infographic-First Visualization

---

# Status

Accepted

---

# Date

2026-06-03

---

# Context

Large tables create cognitive fatigue.

Users understand visual patterns faster.

---

# Decision

Visualization priority:

1. Infographics
2. Heatmaps
3. Capital Flow Maps
4. Charts
5. Tables

---

# Examples

Sector Rotation Maps

Theme Leadership Maps

Capital Flow Maps

Portfolio Health Maps

---

# Benefits

* Faster Understanding
* Better Engagement
* Improved Retention

---

# Long-Term Outcome

Platform becomes insight-centric rather than spreadsheet-centric.
